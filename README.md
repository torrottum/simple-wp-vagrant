# Simple WordPress Vagrant
A Vagrant setup for WordPress with Ubuntu 14.04 (Trusty), Nginx, MySQL, and PHP.

## Features
* Nginx mainline
* php-fpm 5.5.x
* MySQL 5.5.x
* WP-CLI
* git
* Fast site creation (on every `vagrant up`)
